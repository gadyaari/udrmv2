Run those commands in the project directory.
After this you should get directory named: "node_modules" in udrmv2 folder.
Make sure to have node version 6 or above (ECMA 6). When install couchbase make sure to have c++11 compiler.

npm install -g mime
npm install mime
npm install follow-redirects
npm install mkdirp
npm install node-ini
npm install couchbase
npm install nodeunit -g
npm install bluebird
npm install xml2js
